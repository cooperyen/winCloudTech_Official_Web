WinCloudTech

透過 WinCloudTech 實體主機自動化佈署，主機上機櫃開機後自動安裝作業系統及軟體，透過管理介面直接設定，不必一台台辛苦安裝，節省管理人力及成本。

# Modern

功能特色

強化雲端架構

可擴展之雲端服務架構

本平台架構可以根據使用需求，隨時擴充服務規模。

快速安裝部署

透過簡單幾步驟，省去繁雜安裝指令，快速部署及大量佈建更快更輕鬆。

輕量化安裝

非必要之程式套件不進行安裝，最低一台即可運作本台系統。

精進服務流程

自助管理介面

使用者可透過本平台介面進行管理及操作，無須執行任何系統指令。

標準API程式界接

可整合企業內部流程系統或是即時通訊系統，針對平台環境相關訊息可即時通知處理。

滾動式版本更新

平台進行程式或核心更新時，無須將運行中的虛擬機停機，即可完成。

強化數位韌性

應用服務不中斷

提供虛擬機高可用及容錯技術，當主機故障時，根據系統設定參數不同，提供相對應的服務水準，提升數位韌性。

雲端資源可靈活應變

依據目前環境使用狀況，智慧化配置資源所屬位置及自動調整資源。

設備即時監測

可監控看各項設備資源使用狀況及異常告警通知，發現問題即時因應。

# Smart System

雲端資源分派與動態調度

SmartProvision

依據用戶、虛擬機模板定義虛擬機分派策略，找出最適合的實體伺服器啟用虛擬機。

LiveMigration

將運行中的虛擬機動態遷移到最適合的實體伺服器。

AutoScaling

依據虛擬機負載變化，動態增加或減少虛擬機數。
